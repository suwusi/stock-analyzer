# 股票池结构

这个 skill 把本地状态持久化到 `data/watchlists.json`。

## `data/watchlists.json`

```json
{
  "watchlist": [
    {
      "ticker": "000001",
      "name": "平安银行",
      "market": "sz",
      "bucket": "watchlist",
      "concept_sector": "银行",
      "tags": ["低波", "财报观察"],
      "note": "等待财报后再评估",
      "cost_basis": null,
      "position_pct": null,
      "target_price": 12.5,
      "stop_loss": 10.8,
      "updated_at": "2026-04-21T10:00:00"
    }
  ],
  "positions": []
}
```

### 每条记录的字段

- `ticker`：6 位 A 股代码
- `name`：股票简称
- `market`：`sh`、`sz` 或 `bj`
- `bucket`：`watchlist` 或 `positions`
- `concept_sector`：手工指定或自动补全后的概念板块标签
- `tags`：字符串标签列表
- `note`：自由备注
- `cost_basis`：数字或 `null`
- `position_pct`：仓位百分比，使用数字表示，例如 `20`
- `target_price`：数字或 `null`
- `stop_loss`：数字或 `null`
- `updated_at`：ISO 时间戳

板块信息不再单独保存到第二个 JSON 文件。后续分析读取的人工板块信息，统一来自 `watchlists.json` 里的 `concept_sector`。
